* Ghostty Terminal Configuration
:PROPERTIES:
:ID:       38ee36dc-396b-4430-9b8c-e121f9a53932
:END:
This file contains the configuration settings for the Ghostty terminal emulator. It's written using Org
Mode's literate programming features, allowing for clear documentation alongside the configuration code.

The configuration is automatically tangled (extracted) into a file named =config= in the same directory
whenever this Org file is saved in Emacs, thanks to the =#+auto_tangle: t= property and a custom Emacs
Lisp function (detailed at the end). This ensures the Ghostty application always uses the latest settings
defined here.

** Basic Settings
:PROPERTIES:
:ID:       a8db8b74-02c0-41d8-bf1b-6fa1e9cb17f4
:END:
This section covers fundamental appearance and behavior settings for the terminal window.

#+begin_src conf
#########
# Basic #
#########
# Ghostty only supports full-line comments (starting with '#').
# Inline/trailing comments are NOT allowed, so each note lives on its own line.

# Use Ghostty's custom titlebar instead of the GTK default.
gtk-titlebar = false
# Set the terminal background transparency (0.0=fully transparent, 1.0=fully opaque).
background-opacity = 0.80
# Set the TERM environment variable value reported by the terminal.
term = xterm-256color
# Disable confirmation dialog when closing a terminal window/tab.
confirm-close-surface = false
# Disable window decorations managed by the window manager. Useful for tiling WMs.
window-decoration = none

# Set the default font size.
font-size = 12
# Specify the primary font family. 'MesloLGS NF' includes Nerd Fonts glyphs.
font-family = MesloLGS NF
# Set the window class hint, useful for window manager rules.
class = ghostty
# Apply a slight thickening effect to the font for better visibility.
font-thicken = true
#+end_src

** Scrolling
:PROPERTIES:
:ID:       a1b2c3d4-1111-4aaa-9bbb-cccc22222222
:END:
Controls mouse-wheel and touchpad scrolling behavior.

Ghostty reports wheel events to mouse-aware TUIs (like OpenCode) and sends
=mouse-scroll-multiplier= wheel events per tick. Alacritty, by contrast, ignores
its =scrolling.multiplier= while an app captures the mouse, so the same app can
feel much faster in Ghostty. Keep =discrete= low (1) for TUIs and tune =precision=
for touchpad feel.

- =mouse-scroll-multiplier=: =precision:<float>,discrete:<int>=. Defaults are
  =precision:1,discrete:3=. =discrete:1= sends one event per wheel tick;
  =precision:0.5= halves touchpad sensitivity.

#+begin_src conf
##############
# Scrolling  #
##############

# Slow down mouse/touchpad scrolling (Alacritty used multiplier 3; Ghostty
# defaults to discrete:3, precision:1, which feels too fast on a touchpad).
mouse-scroll-multiplier = precision:0.5,discrete:1
#+end_src

* Setting Up Auto-Tangle
:PROPERTIES:
:ID:       fae42d41-2163-436c-b0ed-7601e6682929
:END:

This configuration file uses Org Mode's literate programming capabilities. To automatically generate the
=config= file needed by Polybar every time this =.org= file is saved in Emacs, you need to add the
following Elisp code to your Emacs initialization file (e.g., =~/.emacs.d/init.el=). This code defines a
function `org-babel-auto-tangle` and adds it to the `after-save-hook`. When an Org file is saved, this
function checks for the `#+auto_tangle: t` property at the beginning of the file and, if found, runs
`org-babel-tangle` to generate the output file(s) (in this case, =config=).

#+begin_src emacs-lisp :tangle no
;; Auto-tangle configuration files
(use-package org
  :config
  (defun org-babel-auto-tangle ()
    "Automatically tangle org files when saved if they have '#+auto_tangle: t'."
    (when (eq major-mode 'org-mode)
      ;; Check for the #+auto_tangle property in the Org file's keywords
      (let ((auto-tangle (cdr (assoc "auto_tangle" (org-collect-keywords '("PROPERTY"))))))
        ;; If the property exists and is set to "t", tangle the file
        (when (and auto-tangle (string= auto-tangle "t"))
          (org-babel-tangle)))))
  ;; Run the auto-tangle function after any buffer is saved
  (add-hook 'after-save-hook #'org-babel-auto-tangle))
#+end_src
