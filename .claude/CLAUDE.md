# GSD Patch Instructions

Canonical GSD patch instructions live at:

- `~/.config/gsd-patches/CLAUDE.md`

Canonical patch changelog lives at:

- `~/.config/gsd-patches/gsd-customizations.md`

Edit only canonical files in `~/.config/gsd-patches/`, then run:

```bash
~/.config/gsd-patches/bin/sync all
~/.config/gsd-patches/bin/check all
```
